# This is a POC for developing a prediction model using the Chaos Theory and AT-LSTM model

- Python 3.8 or above
- Jupyter Notebook
- Numpy
- Pandas
- TensorFlow

- Twitter API

- News portals, scrap data

- [Nepal Rastra Bank Website](https://www.nrb.org.np/)
